== Installation ==

# Include the dmg-challenge folder inside of the wp-content/plugins/
# Activate the plugin
# In the Gutenberg editor, add the new block "dmg Search Link"
# In the WP-CLI a command like this can be used: wp dmg-read-more search --date-before=2025-10-01 --date-after=2025-09-01
